# 使用说明 · seat-fitting

面向 SeAT 管理员、FC、普通成员的操作手册。安装请看 [README.md](README.md)，技术细节看 [CLAUDE.md](CLAUDE.md)。

---

## 1. 核心概念

| 概念 | 是什么 | 例子 |
|---|---|---|
| **配装（Fitting）** | 一套具体的舰船 + 模组 + 弹药，对应一段 EFT 文本 | "027重攻海狞" |
| **配装分组（Fitting Group）** | 多个配装的集合，UI 上叫"配装分组"（底层叫 doctrine） | "027重攻海狞"包含 T1 / T2 两套 |
| **入门 / 进阶** | 同一配装的两档技能门槛 | 入门=能开起来，进阶=开得稳/输出足 |
| **辅助技能方案** | 一份独立的"技能 + 等级"清单，可挂到配装或分组上 | "工程全4级" |
| **检查上下文** | "我现在是按哪个分组检查这条 fit"——同一 fit 在不同分组里可以有不同的辅助方案 | 同一艘船在出门散打和本地反收割里要求不同 |

## 2. 五种角色

按 SeAT 权限分（划分都是 `military`）：

| 权限 | 谁用 | 能做什么 |
|---|---|---|
| `fitting.view` | 全员 | 个人配装检查 |
| `fitting.create` | 管理员 / 教官 | 录入/编辑/复制/重命名配装、维护技能要求、维护辅助技能方案 |
| `fitting.doctrineview` | 全员或骨干 | 浏览配装分组（含技能检查） |
| `fitting.reportview` | 军团检查者 | 公司/联盟范围批量技能检查 |
| `fitting.fleet_review` | 舰队长 / 现场指挥 | 实时舰队成员技能审查 |
| `fitting.lock_doctrine` | 高层管理员 | 锁定/解锁配装分组防误改 |
| `fitting.manage` / `fitting.corporation_report` | 预留 | 当前未直接挂在路由上 |

## 3. 典型使用流程

### 流程 A：管理员录入一个配装

**目标**：把游戏里设计好的"027重攻海狞"放进系统。

1. 在 EVE 客户端：装备 → 你的配装 → 右键复制 EFT。
2. SeAT 进入 **配装及技能管理**。
3. 点 "Add Fitting"，把 EFT 文本粘进去。
4. （可选）在右侧填入 DPS / DPH：
   - **入门 DPS / DPH**：基础技能档预期能打出来的伤害
   - **进阶 DPS / DPH**：满档时能打出来的伤害
5. 保存。系统会：
   - 自动解析 EFT，把模组放进对应槽位
   - **递归算出 minimum 技能要求**（包括所有前置技能链）
6. 出现在左侧配装列表后，点该配装：右侧弹出 requirements editor。
7. **点 fit 详情里的某个装备**（比如点击"重型导弹发射器 II"）→ 右侧编辑器自动筛选，只显示这件装备相关的技能。
8. 调整入门/进阶的具体技能等级；点保存。

**思路提示**：
- 自动算出来的会列出几十上百条技能，那是 SDE 完整链路。**不用全部手动确认**——没改动的行系统会自动维护，重导 EFT 时跟着刷新。
- 你只需要把真正想"卡门槛"的几条改一下（比如把"加达里巡洋舰 III"提到 V），这些行会被标成手动，系统不再覆盖。

### 流程 B：管理员组织一个配装分组

**目标**：把"027重攻海狞"做出来，包含 T1 入门版和 T2 主力版。

1. 进入 **配装分组**。
2. 点 "Add Doctrine"，命名"凤凰组"。
3. 把已有的 `027海狞 T1` 和 `027海狞 T2` 两个配装拖进去。
4. （可选）给整个分组挂一份辅助技能方案，比如"凤凰组通用技能 IV"——组里两条 fit 都要满足这份方案。
5. （可选）也可以**只挂到 T2 fit 在这个组里**——比如要求"T2 主力舰特有的辅助技能"，只在 T2 上生效，T1 不受影响。
6. 配装稳定后，点锁定（需要 `fitting.lock_doctrine`），防止误改。

**思路提示**：
- **同一艘船、不同装备 → 建独立 fit 放同一分组**（T1 和 T2 模组用的物品 ID 不同，必须分两条 fit）
- **同一套装备、不同档技能要求 → 一条 fit 用入门/进阶两档**
- 判断口诀：**换装备了吗？换了 → 两条 fit；没换 → 一条 fit 两档**

### 流程 C：成员自查能不能开

**目标**：成员 Alice 想知道自己能不能开"凤凰号-T2"。

1. 进入 **个人配装检查**。
2. 顶部选角色（多账号用户切换主号/小号）。
3. 左侧树展开"027出门散打" → 点击 `027重攻海狞`。
4. 右侧显示：
   - **入门 / 进阶** 两个 tab，分别有 ✓ 或 ✕ 角标
   - 不达标的技能逐条列出，配等级条
   - 当前生效的辅助技能方案列出

**思路提示**：
- 在"凤凰组"下点的 T2，和在"援护组"下点的同一艘 T2，**检查结果可能不同**——因为不同分组挂的辅助方案不同。
- 这就是"按分组上下文检查"的设计。

### 流程 D：军团/联盟批量检查

**目标**：发"凤凰组"之前 FC 想知道全公司有多少人能开。

1. 进入 **军团技能检查**。
2. 选联盟、公司（多选）。
3. 选配装分组（必选）—— 例如"027出门散打"。
4. （可选）筛选某一条 fit—— 例如只看 T2。
5. 出报告：
   - **每个 SeAT 主账号一行**（不是按角色，多 alt 折叠到主号）
   - 每个 fit 列三档：能驾驶舰船 / 入门通过 / 进阶通过
   - 顶部汇总：达标人数 / 比率

**思路提示**：
- 比率按"账号"算，不按"角色"算。一个有 5 个小号的玩家算一个人，不算 5 票。
- 不达标的玩家有"导出未达标名单"功能，方便发 Discord 通知。

### 流程 E：舰队实时审查

**目标**：FC 集结完想现场过一遍每个人开的船、技能能不能跟上输出预期。

1. 舰队长在 EVE 里**已经组建好舰队**（必须是开舰队的本人在用 SeAT）。
2. 该角色 SeAT 上必须授权了 `esi-fleets.read_fleet.v1` 的 SSO scope。
3. SeAT 进入 **舰队审查**。
4. 选配装分组（例如"027出门散打"）。
5. 系统自动识别当前舰队，对每个成员：
   - 用其当前开的船型匹配分组里的 fit
   - 列出该成员对该 fit 的入门/进阶检查结果
6. 顶部三张概览卡：
   - **整队预估 DPS / DPH**（按全员实际状态加权）
   - **理论上限 DPS / DPH**（按全员都达到入门基线）
   - **状态分布**：通过 / 入门 / 进阶 / 未审查
7. 没匹配到任何 fit 的成员（开了不在分组里的船型）单列在"未审查"。

**思路提示**：
- DPS/DPH 的录入是手填的（流程 A 第 4 步），系统不会自己算。**没填的 fit 在舰队审查里显示 `—`**。
- 报告说明里特别强调：**检查人需登录舰队长账号、DPS 按入门基线、不要按太快**——是为了避免对没刷新 ESI 缓存的成员误判。

## 4. 设计思路（FAQ）

**Q：为什么"入门最低要求"会列出几十条技能？**
是按 SDE 递归算出来的完整技能链，包括"驾驶 → 子系统 → 武器 → 弹药 → 电子战"层层前置。99% 都是你不关心的基础技能，把你真正想卡的那 3-5 条改高就行，其他行交给系统自动维护。

**Q：为什么改一艘船的技能要求，另一艘船没受影响？**
系统的技能要求是按 **每个 fit** 存的，不是按舰种存的。同一艘船的两个 fit 各自有独立的技能要求。

**Q：我想做"基础版"和"强化版"两套，建一条 fit 用两档，还是建两条 fit？**
判断标准是**装备是否相同**：
- 装备相同（只是希望技能更强，比如工程学补强或者导航学补强）→ **一条 fit，用入门/进阶两档**
- 装备不同（T1 vs T2 模组）→ **建两条 fit，放进同一个分组**

**Q：辅助技能方案挂在分组和挂在 fit 上有什么区别？**
- 挂分组：组内所有 fit 都要满足，且**只在那个分组的检查里生效**
- 挂 fit（不指定分组）：宇宙级——这条 fit 在任何分组里、单独检查时都要满足
- 挂 fit（指定分组）：**只在那个分组里检查这条 fit 时才生效**——同一 fit 在分组 A 和分组 B 里可以挂不同的方案，互不串扰

**Q：导入 EFT 后能再次编辑同一条 fit 吗？**
能。重新粘贴新的 EFT 时，在左侧选中要替换的那条 fit，再保存。**系统会保留你之前手动调过的技能要求**——只有 `calculated`（系统自动算出）的那部分跟着更新。

**Q：清空一条 fit 的所有"入门技能要求"为什么没用？**
当前实现：清空后系统会自动按 SDE 重新生成一份基础要求作为兜底。**没有"这条 fit 不做最低检查"这个选项**。如果想让一条 fit 完全不参与检查，最好的办法是不把它放进任何分组里。

## 5. 容易踩的坑

| 坑 | 表现 | 怎么办 |
|---|---|---|
| 浏览器缓存旧 JS | 升级后新功能没出现 | `Ctrl+Shift+R` 强刷 |
| 手动调过的技能不自动刷新 | 改了 EFT 重导，某些技能等级没跟着更新 | 这是设计：手动调过的不会被覆盖。需要刷新就把那条删掉，重存时它会变回自动管理 |
| 多账号比率反直觉 | 报告里 100 个角色但比率分母只有 30 | 比率按**主账号**算，不按角色算 |
| 舰队审查找不到舰队 | "未在舰队中"错误 | 必须由**当前舰队长**的角色登录 SeAT；该角色必须授权过 `esi-fleets.read_fleet.v1` |
| 装备点击筛选不出现 | 点装备没反应 | 必须在**配装及技能管理**页（不是个人检查页），且右侧编辑器已展开 |
| DPS/DPH 显示 `—` | 配装详情没数字 | 流程 A 第 4 步没填，回去补 |
